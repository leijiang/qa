## 一切都是对象
* [《一切都是对象》](https://blog.csdn.net/wang_chaunwang/article/details/79915755)
    * 主要是一种思想,面对对象的思想,区别与面向过程而言,面向对象三个特征:封装(逻辑对外不可见,) 继承(儿子继承老子) 多态 (这个有点复杂,就理解为程序运行时候的动态绑定吧)
